# EMF 2024 dome

AKA the "75%" dome

## What is it

All the parts required to cut hubs for a 2V dome out of 6mm MDF.

## What to Cut

For a 2V dome:

| Part  | Count |
| 5-way hubs   | 6        |
| 6-way hubs   | 10       |
| 4-way hubs   | 10       |
| A-connectors | 60       |
| B-connectors | 70       |
